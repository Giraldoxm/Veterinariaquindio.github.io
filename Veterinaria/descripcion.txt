.container-fluid: Esta clase se utiliza para crear un contenedor de ancho completo que se extiende a lo largo de toda la pantalla.

.bg-color-info: Establece el fondo del elemento con un color específico. En este caso, el color es informativo.

.font-color-general: Define el color del texto dentro del elemento. En este caso, se utiliza para establecer el color del texto general.

.row: Define una fila dentro de un contenedor. Se utiliza para organizar elementos en filas.

.col-12 y .col-lg-6: Clases para definir el comportamiento de las columnas en el sistema de cuadrícula de Bootstrap. .col-12 indica que la columna ocupará las 12 columnas en todos los tamaños de pantalla, mientras que .col-lg-6 indica que en pantallas grandes ocupará 6 columnas.

.text-center y .text-lg-left: Alinea el texto en el centro o a la izquierda, respectivamente.

.navbar: Define una barra de navegación.

.navbar-toggler: Clase para el botón de hamburguesa en la barra de navegación.

.navbar-expand-lg: Expande la barra de navegación en pantallas grandes.

.sticky-top: Fija la barra de navegación en la parte superior de la pantalla mientras se desplaza hacia abajo.

.brand-name: Establece estilos para la marca del sitio en la barra de navegación.

.collapse: Oculta los elementos en pantallas pequeñas.

.ml-auto: Alinea un elemento a la izquierda en pantallas pequeñas y a la derecha en pantallas grandes.

.bg-img-principal: Establece un fondo de imagen para la sección principal.

.d-none y .d-lg-block: Oculta el contenido en pantallas pequeñas y lo muestra en pantallas grandes.

.hrblanco: Establece un estilo específico para una línea divisoria (hr).

.align-items-center: Alinea los elementos del contenedor verticalmente en el centro.

.img-fluid: Hace que las imágenes sean responsive para adaptarse al tamaño del contenedor.
